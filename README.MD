# Inventory Management System (Java)

## 📌 Overview
This project is an advanced Inventory Management System using Java Collections Framework.

## 🔧 Features
- Unique product storage using HashSet
- Sorted product display using TreeSet & Comparators
- Transaction history using LinkedList
- Undo operation using Stack
- Low stock alerts using Queue
- Search functionality using Generic methods

## 🧱 Technologies Used
- Java
- Collections Framework

## ▶️ How to Run
1. Compile:
   javac -d . src/com/inventory/**/*.java

2. Run:
   java com.inventory.Main

## 📊 Data Structures Used
| Structure | Purpose |
|----------|--------|
| HashSet | Unique SKU |
| TreeSet | Sorted Data |
| LinkedList | Transactions |
| Stack | Undo |
| Queue | Low Stock |
